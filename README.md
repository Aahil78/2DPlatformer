# 2DPlatformer - A simple 2D Plaformer Game!

Controls:

W / Up Arrow / RShift / LShift / Space - Jump

A / Left Arrow - Left

D / Right Arrow - Right

Collect the 2 keys on the first level, the tunnel will be unlocked, then go through the tunnel, do double jumps to go to each platform. You will then encounter a wall, simply hold D and spam jump to climb the wall, then dodge the enemies to reach your final reward!
